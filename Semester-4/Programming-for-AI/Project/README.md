## 📌 Project Overview

The **Drug Toxicity Prediction System** is a Machine Learning project developed to predict whether a drug is likely to be **toxic or non-toxic** based on its input features. The objective is to demonstrate the application of machine learning techniques in healthcare and drug discovery, where predictive models can assist researchers in identifying potentially harmful compounds at an early stage.

This project was completed as part of the **Programming for AI** course during my BS Artificial Intelligence program.

---

## 🎯 Objectives

* Predict the toxicity of drug compounds using machine learning.
* Perform data preprocessing and feature preparation.
* Train and evaluate a classification model.
* Analyze model performance using evaluation metrics.
* Demonstrate the practical use of AI in healthcare applications.

# 🚍 Project Overview

The **Smart City Transport Management System** is a Java-based desktop application developed to efficiently manage public transportation services within a smart city. The system simplifies the management of buses, routes, drivers, passengers, schedules, and ticket information by integrating a **MySQL relational database** with a user-friendly Java interface.

This project was developed using **Java (NetBeans IDE)** and **MySQL** as part of the **Database Systems** course during my **BS Artificial Intelligence** program.

---

# 🎯 Project Objectives

* Develop a complete transport management system using Java.
* Design and integrate a relational MySQL database.
* Perform CRUD (Create, Read, Update, Delete) operations.
* Manage transport records efficiently.
* Demonstrate practical implementation of Database Management System concepts.

# ⚙️ How It Works

1. Launch the Java application.
2. Connect to the MySQL database using JDBC.
3. Select the desired management module.
4. Perform Create, Read, Update, or Delete operations.
5. Store and retrieve data securely from the database.

---

# 💻 Installation & Setup

### Clone the repository

```bash
git clone https://github.com/Atika-Mughal/Smart-City-Transport-Management-System.git
```

### Requirements

* Java JDK 8 or above
* Apache NetBeans IDE
* MySQL Server
* MySQL Connector/J (JDBC Driver)

### Database Setup

1. Open MySQL.
2. Create a new database.
3. Import the provided `.sql` file.
4. Update the database username and password in the Java project if required.
5. Run the project from NetBeans.

# 🌍 Project Overview

The **AI-Based Smart City Network with Disaster Recovery** is a Computer Networking project that simulates a resilient and intelligent smart city communication infrastructure. The network is designed to ensure reliable connectivity between different city departments while incorporating disaster recovery strategies to maintain communication during network failures.

The project demonstrates how networking concepts, redundancy, and AI-inspired monitoring approaches can improve the reliability, scalability, and fault tolerance of modern smart city environments.

This project was developed using **Cisco Packet Tracer** as part of the **Computer Networking** course during my **BS Artificial Intelligence** program.
